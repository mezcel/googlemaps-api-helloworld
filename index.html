Instructions and Waypoints

Generate a .csv file with your desired way-points. Restrict the list to less than 20 locations. Each record should contain data in the following order: "title", "address", "url", "phone", "email", "purpose", "hours", "about". It is recommended that you use a spreadsheet and export a .csv file which is delineated by only commas separating each field.

To slim down the display of points you can group the .csv by state or flags to focus your travels.
